# TypeScript
TypeScript讲解

1. 开发环境搭建
    1. 下载node.js 安装
    2. 安装typescript  npm i -g typescript  通过tsc查看是否安装
    3. 创建一个ts文件
    4. 执行 tsc xxx.ts 就可以将ts文件编译成js文件，在html中引用后可以在浏览器中访问
